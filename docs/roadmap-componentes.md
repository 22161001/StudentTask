# Roadmap de componentes y funciones

## Componentes ya implementados
- `ProtectedRoute`: protege las rutas privadas del sistema.
- `Header`: muestra contexto de página, fecha y usuario autenticado.
- `Sidebar`: navegación principal del sistema.
- `Card`: tarjeta reutilizable para métricas del dashboard.

## Páginas ya implementadas
- `Login`: validación básica de credenciales demo.
- `Dashboard`: métricas y próximas entregas.
- `Subjects`: CRUD visual de materias con localStorage.
- `Tasks`: CRUD visual de tareas con filtros y cambio de estado.
- `Settings`: perfil y configuración general.

## Funciones nuevas recomendadas para el siguiente avance
1. Conectar login y CRUD a una API REST en PHP.
2. Sustituir localStorage por peticiones a MariaDB.
3. Agregar alertas de validación más completas.
4. Crear módulo de alumnos para administrador.
5. Implementar búsqueda de tareas por título y materia.
6. Agregar vista calendario por día, semana y mes.
7. Integrar recordatorios reales desde backend.
8. Agregar recuperación de contraseña.
9. Implementar tablas paginadas y filtros por prioridad.
10. Crear reportes de tareas pendientes y completadas.
